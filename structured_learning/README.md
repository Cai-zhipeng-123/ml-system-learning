# 结构化学习（Structured Learning）

利用图结构、概率依赖、因果关系和时空关系建模复杂世界。

## 学习范式

- [图学习（Graph Learning）](./graph_learning/): 利用图结构传播节点关系
- [图表示学习（Graph Embedding）](./graph_embedding/): 学习节点或图的低维表示
- [概率图模型（Probabilistic Graph）](./probabilistic_graph/): 用图表示随机变量依赖关系
- [贝叶斯学习（Bayesian Learning）](./bayesian_learning/): 基于后验概率更新知识
- [因果学习（Causal Learning）](./causal_learning/): 学习变量间因果关系
- [时空学习（Spatio-temporal Learning）](./spatio_temporal_learning/): 联合建模空间与时间依赖
- [序列建模（Sequential Learning）](./sequential_learning/): 建模时间顺序依赖
- [关系学习（Relational Learning）](./relational_learning/): 学习实体与关系交互
