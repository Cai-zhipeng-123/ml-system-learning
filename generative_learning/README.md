# 生成式学习（Generative Learning）

关注建模数据分布 P(X) 或联合分布 P(X,Y)，用于生成、补全和表征学习。

## 学习范式

- [概率生成学习（Probabilistic Generative）](./probabilistic_generative/): 建模数据概率分布并生成样本
- [自回归生成（Autoregressive Modeling）](./autoregressive_modeling/): 按条件概率逐步生成序列
- [变分生成学习（Variational Learning）](./variational_learning/): 学习潜在变量概率分布
- [对抗生成学习（Adversarial Learning）](./adversarial_learning/): 生成器与判别器对抗训练
- [扩散生成学习（Diffusion Learning）](./diffusion_learning/): 学习逐步去噪恢复数据分布
- [掩码生成学习（Masked Modeling）](./masked_modeling/): 预测被遮蔽部分以恢复原数据
- [能量模型（Energy-based Learning）](./energy_based_learning/): 用能量函数刻画数据分布
- [世界模型（World Model）](./world_model/): 学习环境动态与状态转移
- [多模态生成（Multimodal Generation）](./multimodal_generation/): 联合建模多模态数据分布
