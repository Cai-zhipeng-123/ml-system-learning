# 交互式学习（Interactive Learning）

通过环境反馈、人类反馈或主动交互持续优化策略。

## 学习范式

- [强化学习（Reinforcement Learning）](./reinforcement_learning/): 与环境交互以最大化长期奖励
- [模仿学习（Imitation Learning）](./imitation_learning/): 模仿专家行为学习策略
- [主动学习（Active Learning）](./active_learning/): 主动选择最有价值样本进行标注
- [人类反馈学习（RLHF）](./rlhf/): 利用人类偏好优化模型行为
- [赌博机学习（Bandit Learning）](./bandit_learning/): 在探索与利用之间动态权衡
- [逆强化学习（Inverse RL）](./inverse_reinforcement_learning/): 从行为反推奖励函数
