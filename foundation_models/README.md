# 现代基础模型（Foundation Models）

面向大规模预训练、通用表示学习、跨任务泛化和多模态统一。

## 学习范式

- [大语言模型（LLM）](./llm/): 基于海量文本学习通用语言能力
- [视觉基础模型（Vision Foundation Model）](./vision_foundation_model/): 大规模视觉预训练
- [多模态基础模型（Multimodal Foundation Model）](./multimodal_foundation_model/): 联合建模文本、图像、语音
- [Agent Learning（Agent Learning）](./agent_learning/): 基于工具调用与规划执行任务
- [Tool Learning（Tool Learning）](./tool_learning/): 学习调用外部工具解决任务
- [检索增强学习（RAG）](./rag/): 结合外部知识增强生成
- [世界模型（World Foundation Model）](./world_foundation_model/): 学习世界动态规律与规划
- [长上下文学习（Long-context Learning）](./long_context_learning/): 处理超长序列与记忆
