# 判别式学习（Discriminative Learning）

关注从输入到输出的映射，学习决策边界或条件概率 P(Y|X)。

## 学习范式

- [监督学习（Supervised Learning）](./supervised_learning/): 利用标签数据学习输入到输出映射
- [半监督学习（Semi-supervised Learning）](./semi_supervised_learning/): 利用少量标签与大量无标签数据联合训练
- [自监督学习（Self-supervised Learning）](./self_supervised_learning/): 从数据自身构造监督信号学习表征
- [多任务学习（Multi-task Learning）](./multi_task_learning/): 多个相关任务共享表示并联合优化
- [在线学习（Online Learning）](./online_learning/): 数据流式到达时持续更新模型
- [对比学习（Contrastive Learning）](./contrastive_learning/): 拉近正样本、拉远负样本以学习表示
- [排序学习（Learning to Rank）](./learning_to_rank/): 直接优化样本排序关系
- [度量学习（Metric Learning）](./metric_learning/): 学习距离空间中的相似性度量
- [结构化预测（Structured Prediction）](./structured_prediction/): 预测具有结构依赖的输出
- [异常检测（Anomaly Detection）](./anomaly_detection/): 学习正常模式并识别异常样本
