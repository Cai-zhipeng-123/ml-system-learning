# 迁移与泛化学习（Transfer & Generalization）

提升模型泛化、少样本适应和跨领域迁移能力。

## 学习范式

- [迁移学习（Transfer Learning）](./transfer_learning/): 将源任务知识迁移到目标任务
- [元学习（Meta Learning）](./meta_learning/): 学习如何快速适应新任务
- [持续学习（Continual Learning）](./continual_learning/): 连续学习多个任务并避免遗忘
- [领域泛化（Domain Generalization）](./domain_generalization/): 学习跨域稳定特征
- [小样本学习（Few-shot Learning）](./few_shot_learning/): 用极少样本完成新任务
- [零样本学习（Zero-shot Learning）](./zero_shot_learning/): 不见样本直接泛化到新类别
- [参数高效迁移（PEFT）](./peft/): 用少量参数完成模型迁移
- [联邦学习（Federated Learning）](./federated_learning/): 分布式隐私保护协同训练
